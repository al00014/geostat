Requirements

This course requires:

  • a Unix shell environment
  • a working Python installation with a number of extra packages
  • an installation of QGIS with Python support.

The exact details of setting this up will depend mostly on your operating system, but also on other things you may have installed.

The Enthought Canopy version of Python might be a solution for you.

Downloads

From github

Everything is in a github-pages branch on github. You can check things out online there.

Or you can download a Zip archive of the current files.

Some tools are required to build the HTML pages from the source, but you shouldn't need to do that.
